# Frontend (Next.js)

This is a minimal Next.js app that requests a LiveKit token from the backend and demonstrates a simple connection using `livekit-client`.

Setup

```bash
cd frontend
npm install
npm run dev
```

Configuration

Set `NEXT_PUBLIC_LIVEKIT_URL` in your environment to your LiveKit WS/Gateway URL, for example `wss://livekit.example.com`.
